# Welcome to Markdown Viewer

A clean, minimal markdown reader with a beautiful black & white theme.

## Features

- **Clean rendering** — Focus on your content, not distractions
- **File association** — Double-click any `.md` file to open it
- **Recent files** — Quick access to your previously opened documents
- **Drag & drop** — Just drop a markdown file onto the window
- **Syntax highlighting** — Code blocks are beautifully rendered

## Code Example

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
